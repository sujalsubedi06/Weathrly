# 🌦️ Weathrly – Weather Dashboard (Student Project)

**Weathrly** is a modern weather web application built as a student project.  
It fetches real-time weather data and displays it in a clean, glassmorphism-style dashboard.

This project helped me practice:

- Working with APIs  
- Writing JavaScript  
- Designing responsive layouts  
- Handling errors properly  
- Improving UI/UX skills  

---

## ✨ Features

- 🔍 Search weather by city name  
- 🌡 Displays temperature, feels-like, humidity, wind, pressure, visibility, and weather description  
- 📅 Shows current date and day  
- 💾 Offline caching (stores recent searches in localStorage)  
- 📱 Fully responsive design (works on mobile & desktop)  
- 🎨 Modern glass-style dashboard with smooth animations  

---

## 🛠 Built With

- HTML  
- CSS  
- JavaScript  
- Weather API (via Cloudflare Worker backend)  

---

## 📡 API Used

The app fetches weather data from:
https://php.sujal-subedi96.workers.dev/?cityName={CITY_NAME}

Expected response includes:

- City & country  
- Temperature & feels like  
- Humidity, wind, pressure, visibility  
- Weather description & icon  
- Data timestamp
